Politique de confidentialité

Dernière mise à jour : juin 2026

2. Données collectées

Nous collectons uniquement les données strictement nécessaires au fonctionnement du service :

  • Formulaire de réservation : prénom, matière souhaitée (stockés dans Firebase Firestore)
  • Formulaire de contact : prénom, nom, email, téléphone (optionnel), message (transmis par email via Web3Forms)
  • Avis : prénom, texte de l'avis, note (stockés dans Firebase Firestore)
  • QCM : prénom, classe, score obtenu (stockés dans Firebase Firestore)

Nous ne collectons pas de données sensibles, de numéro de carte bancaire, ni de données relatives à des mineurs sans consentement parental.

3. Finalités du traitement

Les données collectées sont utilisées exclusivement pour :

  • Gérer les réservations de cours et informer le professeur
  • Répondre aux messages envoyés via le formulaire de contact
  • Afficher les avis des élèves sur le site
  • Suivre les résultats des QCM d'entraînement

Aucune donnée n'est utilisée à des fins commerciales, publicitaires ou cédée à des tiers.

4. Base légale

Le traitement repose sur :

  • L'exécution d'un contrat pour les réservations de cours
  • Le consentement pour les avis et les QCM
  • L'intérêt légitime pour les demandes de contact

5. Durée de conservation

Les données sont conservées pendant la durée nécessaire à leur finalité :

  • Réservations : supprimées manuellement par l'administrateur après le cours
  • Avis : conservés tant qu'ils sont affichés sur le site (supprimables via l'espace admin)
  • Scores QCM : conservés 12 mois puis supprimés
  • Messages de contact : conservés le temps de traiter la demande

6. Sous-traitants

Nous faisons appel aux prestataires suivants, chacun disposant de leur propre politique de confidentialité :

  • Google Firebase (Firestore, Authentication) – stockage des données et authentification admin
  • Web3Forms – transmission des messages de contact par email
  • GitHub Pages – hébergement du site
  • Google Fonts – chargement des polices de caractères
